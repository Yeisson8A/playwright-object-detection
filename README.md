# Automatización, detección de objetos y OCR
Proyecto de automatización en Python, usando Playwright para descargar imágenes de un sitio web, detectar objetos usando YOLOv11 y extraer texto mediante OCR.

## Requisitos
- Python 3.8+

### Instalar dependencias
`pip install -r requirements.txt`

### Instalar navegadores requeridos por Playwright

- **Instalar todos los navegadores**: `python -m playwright install`
- **Instalar solo Chromium**: `python -m playwright install chromium`

### Ejecutar scraper
`python .\app.py`
